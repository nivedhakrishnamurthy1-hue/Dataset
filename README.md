This repository contains Python-based data analysis projects using real-world datasets. The projects demonstrate data preprocessing, exploration, and visualization techniques to extract meaningful insights.

Scenarios:

Scenario 1: E-commerce Sales Analysis – Analysis of product categories, sales trends, and missing data insights.

Scenario 2: Hospital Patient Records Analysis – Exploration of patient health metrics, distributions, and patterns.

Scenario 3: Housing Dataset Analysis – Investigation of housing attributes, correlations, and trends for price prediction.

Scenario 4: Customer Personality Analysis – Analysis of customer demographics, income levels, age distribution, and spending behavior to understand purchasing patterns.
